# Task Manager Prototype

This is a simple command-line Task Manager written in Python for the CSC299 final project prototype.

## Features
- Add new tasks
- List all tasks
- Search tasks by keyword
- Stores data in a JSON file (`data/tasks.json`)

1. Open a terminal in the project root:
2. Run the program:
3. Follow the menu prompts to add, list, or search tasks.

## Notes
- Uses only built-in Python libraries (`json`, `pathlib`)
- Works on Windows, macOS, and Linux
- JSON file is created automatically the first time you add a task

   cd csc299-project
2. Run the program:
   python tasks1/cli.py
